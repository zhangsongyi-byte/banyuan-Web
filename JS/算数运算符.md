# 算数运算符

```javascript

//当对非Number类型的值进行运算时，会将其转换为Number类型
//任何值和NaN做运算都为NaN
//对字符串做加法，结果返回字符串
//任何值和字符串做加法运算，都会先转化为字符串再计算


var a = 123 + 223;

var a = true + 1;

var a = '123' + '456';

var a = '123' + 456;

//转化成字符串
var a = 123 + '';

// 从左往右
var a = 1 + 2 + '3';

var a = 1 + '2' + 3;

// 其他运算，都是转换成number
// - * / %
var a = 2 * undefined


//取模 （ 取余数 ）
var a = 9%3

var a = 9%4
```
